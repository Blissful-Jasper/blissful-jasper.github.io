## 配置控制

所有页面都支持通过 `_config.yml` 中的 `page_sections` 配置来控制显示/隐藏：

```yaml
page_sections:
  about: true
  research: true
  publications: true
  journals: true
  blog: true
  links: true
  maps: true
  contact: true
```

当某个页面被禁用时，会显示优雅的禁用提示页面。
